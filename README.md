# GraphAdventure
A graph detective is assigned with a task.Here is a C++ code of the assigned problem which can :

1.Take a graph as input from the console.
2.Show the graph using an adjacency matrix (like the graph's secret map)
3.Find the degree of each vertex (who has the most friends in the graph?)
4.Check if the graph is connected or not (can every vertex reach each other?)
5.Count the total number of edges (how many bridges exist in the city)
