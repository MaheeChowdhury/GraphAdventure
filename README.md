# GraphAdventure
A graph detective is assigned with a task.Here is a C++ code of the assigned problem which can :

- Take a graph as input from the console.
- Show the graph using an adjacency matrix (like the graph's secret map)
- Find the degree of each vertex (who has the most friends in the graph?)
- Check if the graph is connected or not (can every vertex reach each other?)
- Count the total number of edges (how many bridges exist in the city).
